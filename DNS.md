# DNS

DNS (Domain Name System) translates domain names into IP addresses.

Example:

google.com → 142.250.x.x

## Common Record Types

- A Record
- AAAA Record
- MX Record
- CNAME

## Commands

nslookup google.com

ping google.com

## Why DNS Matters

Cybersecurity analysts investigate DNS traffic to identify malicious domains, phishing websites and command-and-control communication.
