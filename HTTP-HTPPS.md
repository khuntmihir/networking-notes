# HTTP and HTTPS

HTTP (HyperText Transfer Protocol) is used to transfer web pages between a client and a server.

## HTTP

Default Port:

80

Characteristics:

- Unencrypted
- Data can be intercepted
- Less secure

Example:

http://example.com

## HTTPS

HTTPS (HyperText Transfer Protocol Secure) encrypts communication using SSL/TLS.

Default Port:

443

Characteristics:

- Encrypted communication
- Protects sensitive data
- Prevents eavesdropping

Example:

https://google.com

## HTTP Request Methods

### GET

Retrieves data from a server.

### POST

Sends data to a server.

### PUT

Updates existing data.

### DELETE

Removes data.

## Why HTTPS Matters

Cybersecurity professionals monitor web traffic and identify insecure communications.

HTTPS helps protect:

- Passwords
- Banking information
- Personal information
- Login sessions
