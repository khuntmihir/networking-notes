# OSI Model

The OSI (Open Systems Interconnection) Model is a framework used to understand how data travels across a network.

## The 7 Layers

### Layer 7 – Application
Provides network services to applications.

Examples:
- HTTP
- HTTPS
- FTP
- DNS

### Layer 6 – Presentation
Formats and encrypts data.

Examples:
- SSL/TLS
- JPEG
- PNG

### Layer 5 – Session
Establishes and manages communication sessions.

Examples:
- NetBIOS
- RPC

### Layer 4 – Transport
Provides reliable or fast data delivery.

Protocols:
- TCP
- UDP

### Layer 3 – Network
Handles routing and logical addressing.

Examples:
- IP
- ICMP

### Layer 2 – Data Link
Handles communication between devices on the same network.

Examples:
- Ethernet
- MAC Addresses

### Layer 1 – Physical
Transmits raw bits through physical media.

Examples:
- Cables
- Switch ports
- Wireless signals

## Why the OSI Model Matters

Understanding the OSI model helps cybersecurity professionals troubleshoot network issues and identify where attacks occur.

Examples:

- Phishing → Layer 7
- DDoS Attack → Layer 3/4
- Packet Sniffing → Layer 2
